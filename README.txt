CodePad is a simple, yet powerful Java text editor.

Current Features:
	1. Ability to write text documents
	2. Line Numbering
	3. Cut, Copy, Paste
	4. Time and Date
	5. An "About" section
	6. New, Open, Save, Save As
	7. Close program
	8. Undo and Redo
Coming soon:

How to run:
	Download JAR file and save it to your desktop. Double-click to run!
